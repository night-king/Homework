# Frontend

这里存放 Homework.today 的前端与可运行原型。

## 子目录

- `parent-web/`：家长端 React + Vite SPA
- `child-web-prototype/`：孩子端高保真 HTML 原型

## 家长端

```powershell
cd frontend/parent-web
npm install
npm run dev
npm run build
```

## 孩子端原型

直接打开：`frontend/child-web-prototype/child-homepage.html`

这个目录当前是原型，不是正式 React 工程。
