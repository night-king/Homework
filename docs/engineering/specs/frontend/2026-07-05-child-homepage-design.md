# 孩子端首页设计稿

- 日期：2026-07-05
- 状态：已完成头脑风暴，已确认首页收口方案，待进入实现
- 范围：仅覆盖孩子端首页，不展开登录、完整 PK 世界页、成就页等二级页面

## 目标

把“每天按家长设置的 schedule 打卡做作业”包装成一个热血、正向、可持续的成长冒险首页，让孩子一进来就愿意开始今天的任务。

## 核心定位

- 首页不是后台，也不是普通待办清单
- 首页是“冒险基地”
- 宠物 / 角色主舞台是视觉中心
- 今日任务区是行动中心
- 喂养反馈是情绪奖励中心
- `Day` 切换是一级导航，用来承接家长按周配置的 schedule

## 首页结构

### 1. 顶部状态条

- 左侧：孩子头像、昵称
- 中间：本周 `Day` 切换条
- 右侧：星星、连击、今日完成度

### 2. 本周 Day 切换条

- 采用 7 个并列徽章，显示 `一 二 三 四 五 六 日`
- 默认高亮今天
- 允许切换查看本周任意一天
- 每天支持以下状态：
  - 今天
  - 未开始
  - 进行中
  - 已完成
  - 休息日

### 3. 宠物 / 角色主舞台

- 占据首页首屏最大面积
- 用于展示当前伙伴的大形象、情绪、成长状态
- 保留一句强提示文案，例如：
  - 今天还差 2 个委托
  - 今天已经全部攻克
  - 今天是自由修整日
- 首页内保留一个小型 `PK 世界` 入口，但不展开完整页面
- 宠物区只强调 3 件事：
  - 宠物本体
  - 当前状态
  - 唯一主操作（如喂养 / 进化）

### 4. 冒险委托板

- 当前选中 `Day` 的任务以平铺卡片展示
- 不做线性闯关地图，不强调先后顺序
- 每张任务卡包含：
  - 任务名称
  - 学科徽章
  - 预计时长
  - 完成状态
  - 主动作按钮

### 5. 即时奖励区

- 任务完成后掉落“食物 / 能量 / 星星”
- 保留一个明确的喂养动作，把“完成作业”和“宠物成长”强绑定
- 用鼓励语强化正反馈

### 6. 成长栏

- 放在首页底部
- 主要展示等级、成长值、连击、勋章位
- 首版以摘要式展示为主，不展开复杂详情

## 宠物进化逻辑

- 宠物采用 `单一路线进化`
- 成长分成两层：
  - `小反馈`：完成单个任务 -> 掉落食物 -> 喂给伙伴
  - `大反馈`：成长值 + 进化晶核满足条件 -> 触发进化
- 成长不回退：
  - 不因某天没完成而降级
  - 不因家长修正记录而让已完成进化回退

### 进化资源

- `食物`：完成任务立即掉落，用于制造即时爽感
- `成长值`：喂养后上升，用于推进长期成长
- `进化晶核`：某一天 100% 完成时获得，用于触发真正的大进化

### 进化规则

- 小进步靠喂养，大进化靠满完成与坚持
- 进化不是自动替换形态，而是出现清晰可点的 `进化` 动作
- 可进化时，这个动作优先级高于普通状态提示

### 进化路线

- `1阶：火苗幼龙`
- `2阶：炽角小龙`
- `3阶：翼焰飞龙`
- `4阶：烈焰守护龙`
- `5阶：星焰龙王`

### 视觉保留点

- 大眼睛
- 暖橙主色
- 浅色肚皮
- 尾焰
- 圆润脸型

目标是让孩子感受到：这是一只“同一只伙伴一路长大”的幼龙，而不是频繁换宠物。

## 宠物状态机

- `修整日`：无任务，伙伴休息、巡逻、晒太阳
- `待命`：今天还没开始，鼓励孩子先完成第一项
- `进行中`：已经推进部分任务，伙伴更有精神
- `待喂养`：奖励已掉落，伙伴明显期待孩子点喂
- `全清完成`：当日任务全部攻克，伙伴庆祝
- `可进化`：成长值和晶核都满足条件，首页高亮进化机会

### 瞬时状态

- `进食反馈`
- `进化演出`

### 状态优先级

`进化演出` > `可进化` > `全清完成` > `待喂养` > `进行中` > `待命` > `修整日`

## 交互原则

- 任务顺序自由，不制造额外流程压力
- 每完成一项任务，立即获得视觉和情绪反馈
- 宠物反馈强于纯数字反馈
- 未完成状态要温和，避免羞耻感和惩罚感
- 休息日也要有内容，不出现空白页

## 视觉方向

- 主题：少年冒险训练营
- 气质：热血、升级、闯关、成长
- 边界：明亮正向，不黑暗、不挫败、不恐吓
- 主要颜色：
  - 熔岩橙 / 胜利金
  - 天空蓝 / 能量青
  - 浅沙色 / 云雾白

## 当前不做

- 完整 PK 世界页
- 成就 / 勋章二级页
- 复杂段位体系
- 登录流程与权限策略
- 真实后端联动

## 原型目标

下一步输出一个独立 HTML 原型，用假数据模拟：

- 周内 Day 切换
- 每日任务板
- 完成任务后的即时反馈
- 伙伴喂养与成长状态

用于和产品一起看页面、继续细调布局和氛围。

## 2026-07-05 首页收口修订

### 目标

在不改变“Day 切换 + 宠物主舞台 + 今日委托 + 喂养循环”核心结构的前提下，收掉首页重复信息，让孩子第一眼先看伙伴，再决定去做任务或喂养。

### 确认方案

- 采用 `A 极简版` 收口方案
- 页面从“多张说明卡”改成“一个主舞台 + 两个功能区”
- 强调宠物自己表达状态，弱化纯文字说明

### 调整后结构

#### 顶部

- 保留 `Day` 切换
- 保留 `今日进度 / 连打 / 星星`
- 不再把这些信息在中部重复展示

#### 左侧主舞台

- 保留宠物主形象
- 保留头顶短状态词，例如：
  - `待命`
  - `待喂`
  - `可进化`
  - `满状态`
- 保留 `进化路线`
- 保留 `伙伴图鉴`
- 保留脚边横向成长槽
- 在成长槽附近仅保留两项硬信息：
  - `Lv`
  - `晶核`

#### 右侧功能区

- `今日委托`
- `补给台`

不再保留独立的：

- `伙伴状态` 卡
- 完整 `今日战力` 四宫格

### 去重原则

- `伙伴状态` 不再单独成卡，而是放进宠物动画和头顶状态词
- `已完成` 不再出现在中部战力板块，因为顶部已有 `今日进度`
- `待喂` 不再出现在战力板块，因为它已经属于 `补给台`
- `状态` 不再重复出现在战力板块和宠物头顶，只保留宠物头顶一处主状态

### 宠物状态表达

首页默认不再出现一整段 `伙伴状态` 说明文案，而是通过动作、表情和能量特效表达：

- `待命`：轻呼吸、眨眼、尾巴小幅摆动
- `进行中`：更精神、尾焰更活跃、身体稍前倾
- `待喂`：期待动作、视线偏向补给方向、脚边轻跳
- `可进化`：全身发光增强、角或翼附近出现能量流、底座亮圈
- `全清 / 满状态`：站姿更舒展，出现胜利粒子

### 补给台设计

- `补给台` 只负责展示奖励与喂养操作
- 空状态只留一句短提示，例如 `去拿补给`
- 有奖励时直接展示掉落物
- 同屏最多正面展示 3 个补给，超出后用 `+N` 角标表示
- 仅保留一个按钮：
  - `喂给伙伴`
  - `喂给伙伴 x2`
- 喂养后做短反馈：
  - 补给飞向宠物
  - 宠物吃到后的反馈动作
  - 成长槽闪动
  - 飘出 `成长+12`

### 最终页面理解

- 上面负责切换今天看哪一天
- 左边负责看伙伴和成长
- 右边负责做任务和喂养

整体目标是让首页更像“孩子的冒险基地”，而不是“带很多说明块的任务面板”。

## 2026-07-05 元素进化修订

### 目标

让“进化”不再只是小配件变化，而是同时带来：

- 龙的身体明显长大
- 龙的形态明显变化
- 新元素能力逐步解锁
- 元素切换带来新的吐息、特效和战斗气质

### 核心原则

- `阶级 = 龙的身体长大`
- `属性 = 龙的战斗气质切换`

也就是说，孩子先感受到“我的龙长大了”，再感受到“我的龙会不同元素了”。

### 主属性体系

首页采用统一的五属性体系：

- `金`
- `木`
- `水`
- `火`
- `土`

`风 / 雨 / 雷 / 电` 不作为独立主属性存在，而是作为不同属性的技能演出与动态语言：

- `金`：金刃、雷电、电弧
- `木`：藤蔓、叶旋、林风
- `水`：水波、落雨、冰息
- `火`：火流、爆焰、火星
- `土`：岩甲、震地、碎石

### 5 阶体型成长目标

#### 1阶：火苗幼龙

- 明显幼态
- 头大身小
- 四肢短
- 翅膀几乎没展开
- 默认解锁属性：`火`

#### 2阶：炽角小龙

- 身体拉长一点
- 角长出来
- 尾焰变大
- 仍然圆润，但已经不是宝宝体型
- 新解锁属性：`水`

#### 3阶：翼焰飞龙

- 第一次明显长大
- 翅膀真正展开
- 腿更长
- 胸口更挺
- 一眼看出“从幼龙长成少年龙”
- 新解锁属性：`木`

#### 4阶：烈焰守护龙

- 体格更厚
- 肩背更强
- 护甲更明显
- 站姿从可爱转向可靠
- 新解锁属性：`土`

#### 5阶：星焰龙王

- 冠、翼展、尾焰拖尾、身体比例全面拉开
- 目标是形成“毕业形态”的王者感
- 新解锁属性：`金`

### 属性解锁节奏

- 1阶：`火`
- 2阶：`火 / 水`
- 3阶：`火 / 水 / 木`
- 4阶：`火 / 水 / 木 / 土`
- 5阶：`火 / 水 / 木 / 土 / 金`

规则：

- 属性解锁后永久保留
- 当前只能激活 1 个属性
- 切换属性不消耗资源
- 不制造“选错属性”的压力，只强调乐趣和表达

### 属性表现语言

#### 火

- 主色：橙红、赤金、深炭
- 主体验：喷火、燃烧、爆焰
- 关键词：冲锋、热血、燃起来

#### 水

- 主色：冰蓝、深蓝、霜白
- 主体验：水息、雨幕、寒潮、冰息
- 关键词：冷静、专注、冻结、流动

#### 木

- 主色：青绿、森白、嫩金
- 主体验：藤蔓、生长、叶风
- 关键词：生长、灵动、治愈、枝叶旋动

#### 土

- 主色：岩棕、赭石、暗金
- 主体验：岩甲、震地、守护
- 关键词：厚重、抗打、守护

#### 金

- 主色：银白、鎏金、电蓝
- 主体验：金刃、雷击、电弧
- 关键词：锋利、王者、爆闪

### 首页属性入口

- 放在宠物右上侧
- 做成围绕伙伴漂浮的“小型元素核心”
- 默认只显示当前属性徽章
- 点击后展开半环形属性选择

状态规则：

- 已解锁属性：可点击
- 当前激活属性：高亮脉冲
- 未解锁属性：灰色封印态 + 小锁

### 属性切换反馈

- 不弹大提示
- 点击后做 0.6 ~ 1 秒短演出
- 宠物周围扫过对应属性光效
- 翅边、尾巴、角、脚下底座同步切换元素风格
- 头顶短暂闪出对应属性名，例如：
  - `水系激活`
  - `金系激活`

### 图鉴展示规则

- `进化图鉴` 仍以 5 阶成长为主轴
- 每个阶段卡片底部新增一行 `本阶新增解锁`
- 已解锁属性显示真实彩色徽章
- 未解锁属性显示灰色封印徽章或 `?`

图鉴中的信息层级：

- 第一层：这只龙长到了哪一阶
- 第二层：这一阶新增了什么元素能力

### 首版原型范围

首版必须做到：

- 5 阶长大明显
- 五属性全部进入体系
- 首页可切换当前属性
- 每种属性有一眼可辨的颜色、特效和气质
- 图鉴能看见每阶新增解锁属性

首版暂不做：

- 属性克制
- 战斗数值
- 独立属性详情页
- 多技能树
- 属性切换资源消耗

### 首版体验目标

孩子应该直接感受到：

- 我的龙真的长大了
- 我的龙真的解锁了新元素
- 切元素之后，整只龙真的变了
