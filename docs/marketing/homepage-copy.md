## 背后的心理学依据（搭建 / 审校参考，不一定上墙）

叙事逻辑是按儿童发展 + 家长心理学校准过的，每一段都站得住：

- **② 痛点 = 逆反心理（reactance）**：外部压力越大，孩子越抵触——精确解释"催了他嫌你烦"，说明催是**反效果**，不是孩子的错。
- **③ 双重赦免**：先卸掉对**孩子**的指责（"不是孩子懒"），再卸掉对**家长**的愧疚（"也不怪你"）——面向"背着愧疚的家长"这一读者，情感 + 信任杀伤力大。核心是**内在动机 vs 外在压力**（自我决定理论：自主·胜任·联结）。
- **④/⑦ 把家长从"外部管控者"退位**：家长是压力源时，孩子永远内化不了动力；把动机搬进游戏，孩子才能发展"自我管理"，同时把亲子关系从"作业权力斗争"里解放出来。
- **⑤ 三层 & "短时间视野"**：小孩对遥远大奖无感，需要**即时反馈（每天星星/成长）+ 近期兑现（阶段小奖）**；并**正面破"过度合理化 / 德西效应"**（物质奖励会削弱内在动机）——讲清主次：内在成长是天天的发动机，家长奖励只是偶尔的里程碑。
- **⑥ 照顾养成 ≠ 贿赂**：为"依赖自己的小生命"付出，长出的是**责任心**；纯代币奖励才养出功利。"养小伙伴"是产品的心理学之魂（Tamagotchi / 照顾者养成）。
